# Annotree 概要

Annotree（Annotate + Tree）是一款专为生成`文件夹目录注释树`而设计的桌面软件，旨在简化技术文档的编写过程。通过直观的可视化操作界面，你可以轻松创建文件夹目录注释树。

系统支持：Mac（M 系列、Intel 系列）、Windows 和 Linux。

> [!NOTE]
>
> 如果你需要生成文件夹目录注释树，一定要试试 Annotree 🌲，它绝对不会让你失望。
